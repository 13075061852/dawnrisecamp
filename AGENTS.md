# Project Preferences

- 本仓库默认用于“本地预览前端，连接云端数据”的工作流。
- 本地前端开发默认连接线上 Worker：`https://dawnrisecamp-api.1308715689.workers.dev`。
- Cloudflare Worker 的 D1 / R2 绑定默认使用远程资源，避免本地与云端数据分叉。
- 结构化、可查询数据优先使用 D1；对象、文件、视频、图片优先使用 R2。
- Worker 代码更新后，应同步部署到云端；数据库迁移更新后，应同步执行远程迁移。
- 前端代码可以先在本地预览，但只有在用户明确确认后，才允许部署到 Cloudflare Pages。
- 前端部署前应先说明将要同步的内容，并等待用户确认。
- 对文本文件默认使用 UTF-8，新增文件优先 LF；编辑现有文件时先保留项目现状。
- 对前端/UI 修改，默认不做浏览器自动验收；完成代码修改后先汇报，由用户自行在浏览器确认并反馈问题。
